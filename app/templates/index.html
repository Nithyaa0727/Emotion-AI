01Computes gradients of the predicted class score w.r.t. last conv feature map
02Weights each channel by its global average gradient
03Applies ReLU — only positive contributions highlighted

How LIME Works

01Segments image into superpixels via SLIC
02Generates 1000 perturbed samples by masking superpixels
03Fits a local linear model to approximate the neural network
04Green supports, Red contradicts prediction
